Glossaire
==========

.. glossary::

   modèle Meta
      Le modèle Meta est le nom du modèle développé dans le cadre de ce projet.

      Il s'agit d'un modèle de prise de décision et de fonctionnement conçu pour être appliqué au sein de groupes.

   groupe
      Un groupe est composé d'un ensemble de membres, réunis pour atteindre un ou plusieurs objectifs.
      Une association, une entreprise, une commune sont différents exemple de groupes, de tailles distinctes.

      Au sein d'un groupe, les membres peuvent avoir des roles et des responsabilités différentes.

   membre
      Les membres d'un groupe peuvent être une personne physique ou morale, chacune d'entre elles
      pouvant être impliquée dans plusieurs groupes simultanément.

   objectif
      Un groupe s'articule autour d'un ou plusieurs objectifs, qui sont les raisons d'être du groupe.

      Les objectifs peuvent être très précis et finis dans le temps, par exemple, une association peut se constituer pour soutenir une personne emprisonnée, mais certains groupes ont des missions plus complexes, qui s'inscrivent dans un temps plus long.

      Ainsi, une entreprise pourrait avoir comme objectif de proposer un ensemble de produits ou de services, en conservant la rentabilité la plus élevée.
